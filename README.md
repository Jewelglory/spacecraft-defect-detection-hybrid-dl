# Hybrid Deep Learning Framework for Automated Spacecraft Component Defect Detection

This project implements a hybrid deep learning architecture for automated surface defect detection using computer vision.

## Project Overview
Manual inspection of spacecraft components is time-consuming and prone to human error. This project proposes an automated defect classification system using a hybrid CNN with channel attention (Squeeze-and-Excitation block).

## Dataset
NEU Surface Defect Dataset

## Models Implemented
1. Baseline CNN
2. Hybrid CNN + SE Attention

## Results
Baseline CNN Accuracy: 91.11%  
Hybrid CNN Accuracy: 92.78%

## Features
- Automated defect classification
- Hybrid deep learning architecture
- Attention-based feature enhancement
- Confusion matrix analysis
